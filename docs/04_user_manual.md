# 4. 用户操作指南 (User Manual)

## 4.1 账号与宝宝设置
1.  **注册与登录**: 使用手机号注册并设置密码。
2.  **创建宝宝**: 首次进入需填写宝宝姓名、性别、生日。
3.  **邀请家人**: 
    -   在“宝宝管理”页面点击“分享”图标。
    -   系统生成专属链接，家人在 24 小时内打开并确认即可加入。

## 4.2 记录操作
-   **Timeline 首页**: 点击右下角悬浮按钮或记录卡片，快速记录单次操作。
-   **喂养**: 记录总量（ml）或左右侧母乳时长。
-   **睡眠**: 点击“正在睡觉”开始计时，结束后确认时长。
-   **统计视图**: 查看历史趋势图，点击图表节点可查看详细数值。

## 4.3 AI 助手使用
-   **智能建议**: 首页顶部显示的“育儿专家建议”卡片。
-   **更新频率**: 建议每记录满 3 天后点击一次“重新生成”，以获得最准确的分析报告。
-   **数据安全**: 所有的健康分析仅基于已记录的结构化数据，不涉及任何生物识别信息。

## 4.4 健康标准参考
-   **生长曲线**: 身高、体重记录会自动标记在 WHO 标准曲线（P3, P15, P50, P85, P97）上，帮助判断宝宝发育是否在正常范围。
