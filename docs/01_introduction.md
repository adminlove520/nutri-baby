# 01. Nutri-Baby 项目简介

Nutri-Baby (育儿助手) 是一款面向新生儿父母的科学育儿辅助工具。项目结合了现代化的前端技术、稳定可靠的后端云服务以及先进的 AI 模型，旨在为每位家长提供最专业、最贴心的育儿支持。

## 🎯 项目目标

-   **记录**: 轻松记录宝宝的每一份饮食、每一次睡眠和每一项成长指标。
-   **洞察**: 通过 AI 分析和数据可视化，让家长看懂宝宝的行为模式和成长曲线。
-   **计划**: 智能化的疫苗接种规划，确保宝宝不遗漏任何一项健康防线。
-   **百科**: 随手可得的科学育儿知识库。
-   **同步**: 图库自动同步到 GitHub，安全备份宝宝成长记录。

## ✨ 核心亮点

### 1. 深度 AI 集成 (OpenAI GPT-4o-mini / MiniMax)
-   **育儿锦囊**: 每日动态生成的科学育儿建议，精准匹配宝宝当前的成长阶段。
-   **健康洞察报告**: 基于喂养与睡眠数据的深度 AI 趋势分析。
-   **接种深度百科**: 告别厚重的纸质指南，随时查询各类疫苗的专业科普与护理建议。
-   **AI 文案生成**: 为分享相册自动生成朋友圈文案。

### 2. 接种管家 (AMap + AI)
-   **全流程覆盖**: 从接种计划制定、百科查询到地理位置导航，实现一站式服务。
-   **一键导航**: 基于高德地图 API，实时显示附近的医院并计算距离。
-   **提醒系统**: 支持站内信实时推送与 QQ 邮箱提醒，确保接种万无一失。

### 3. 成长相册（朋友圈风格）
-   **照片记录**: 记录宝宝每个珍贵瞬间，支持多图上传。
-   **评论点赞**: 家人朋友可以评论和点赞，增强互动。
-   **分享功能**: 生成分享链接，AI 自动生成文案。
-   **GitHub 同步**: 自动将相册同步到 GitHub 仓库，安全备份。

### 4. 家庭共享
-   **邀请机制**: 生成专属邀请链接，家人可轻松加入。
-   **共同记录**: 家庭成员可查看和记录宝宝的全部数据。
-   **实时同步**: 数据变更即时同步，所有家庭成员可见。

### 5. GitHub 图床同步
-   **自动备份**: 图库照片自动同步到 GitHub 仓库。
-   **定时同步**: 支持每天/每周/每月自动同步。
-   **同步策略**: 可选择同步哪些类型的相册。
-   **去重机制**: 智能检测已同步文件，避免重复上传。

### 6. 极速响应与温馨 UI
-   **响应式布局**: 无论是手机平板还是电脑，都能获得一致的流畅体验。
-   **玻璃拟态设计**: 采用 Element Plus 深度定制，带来更符合育儿产品的柔和感官体验。
-   **深色模式**: 支持护眼深色界面。
-   **PWA 离线支持**: 无需安装即可添加到桌面，随时访问。

## 🛠️ 技术概览

-   **前端框架**: Vue 3 + Vite + Pinia + Element Plus
-   **样式体系**: SCSS + Element Plus (Glassmorphism Custom Theme)
-   **后端驱动**: Node.js (Vercel Functions)
-   **数据库层**: PostgreSQL (Vercel Postgres) + Prisma ORM (Accelerate Optimized)
-   **人工智能**: OpenAI (GPT-4o-mini) / MiniMax-M2.7
-   **图床同步**: GitHub REST API
-   **地理位置**: 高德地图 (AMap) JS API 2.0
